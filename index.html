Git for Windows commands:

How to work with this thing:

  1. git status - tracked and untracked files.
  2. git add . - add files in repository.
  3. git commit -m "Initial commit" - prepare files to send.
    1. If type just git commit the VIM IDE will open
    2. Then, you check all the files you need to upload
    3. Press "S"
    4. Type the name of commit in the first line
    5. Press Esc
    6. Type ":wq" in the lowest line
    7. Press Enter
    8. Profit! Commit initialed
  4. git push - upload files to repository

Next, files was uploaded to repository and we can see the changes on GitHub.com
